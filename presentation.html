Security

Security Architecture

7-layer defense-in-depth security implementation

🔐 Authentication

JWT in HTTP-only cookies · SameSite: strict · Secure in production · 24h expiry

🔑 Password Security

bcrypt with 12 salt rounds · Never returned in API responses (select: false)

🛡 Brute-Force Protection

5 failed attempts → 15min lockout · Counter resets on success

📱 Session Management

Max 3 concurrent sessions · Oldest evicted on 4th login · Token blacklisting on logout

🧹 Input Sanitization

MongoDB injection prevention (strips $ and . operators) · XSS prevention (recursive string sanitization)

⏱ Rate Limiting

API-level rate limiter on all /api routes · Separate login & register limiters

🪖 HTTP Headers

Helmet.js — Content-Security-Policy, X-Frame-Options, X-Content-Type-Options, and more
